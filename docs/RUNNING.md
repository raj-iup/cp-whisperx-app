# Running Pipeline Guide

**Execute and manage pipeline jobs**

## Quick Reference
```bash
# Prepare job
./prepare-job.sh in/movie.mp4

# Run pipeline
./run_pipeline.sh --job 20251108-0001

# Check status
./scripts/pipeline-status.sh 20251108-0001

# Resume failed job
./resume-pipeline.sh 20251108-0001
```
