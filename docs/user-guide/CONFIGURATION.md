# Configuration Guide

**Job and system configuration reference**

## Configuration Files

### secrets.json
```json
{
  "TMDB_API_KEY": "your-key",
  "HF_TOKEN": "your-token"
}
```

### .JOB_ID.env
Generated by prepare-job, contains all job settings:
- DEVICE=mps
- WHISPER_MODEL=large-v3
- Stage parameters

See [Architecture](ARCHITECTURE.md) for configuration hierarchy.
